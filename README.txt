This website, which base one HTML CSS JS, is a music player or called "media player" but it is only for audio.

Main idea of this website
- create a music player website

Minimum system requirments

- RAM => 4 GB

Software and IDE requirments
- VScode 1.97 or more
- {Extention} = Live Preview (other preview extension is not recommended)
- Google Chrome browser
